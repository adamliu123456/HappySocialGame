# HappySocialGame（社交对战游戏MVP）

这是一个可快速验证的**社交对战游戏**原型项目，优先实现“原型B：异步回合棋盘突袭”，并预留公会赛季战扩展位。

## 设计目标

- 短局（3-8分钟）与异步回合并存，降低联机压力。
- 公平优先：付费不影响数值，变现以皮肤+通行证为主。
- 社交驱动：好友挑战、观战复盘、公会周赛。
- 可运营：赛季化任务、段位与活动池更新。

## 当前实现（代码层）

- 棋盘与地形系统（平地/森林/堡垒）
- 回合行动：移动、攻击、技能（dash/shield）
- 简化匹配评分：Elo、Glicko RD维护、TrueSkill-like房间质量
- 变现配置校验：抽取概率披露与公平性提示

## 快速开始

```bash
python -m pip install -e .
pytest
```

## 第一阶段里程碑建议

1. **Week 1-2：核心玩法闭环**
   - 单局流程：匹配 -> 选人 -> 回合提交 -> 结算
   - 新手引导：镜像阵容 + 三步教学
2. **Week 3-4：社交功能**
   - 好友挑战、战报分享、观战入口
3. **Week 5-6：赛季化与商业化**
   - 战斗通行证任务系统
   - 皮肤商店与赛季主题活动

## 目录结构

- `src/social_battle/core.py`: 核心规则与评分逻辑
- `tests/test_core.py`: 单元测试
